# Grocery Shop Management System
A management system developed by C programming language in my first semester of undergraduation as a first project in this field

# Intro
 Grocery Shop Management System, is a console- based C program designed to simplify the management of inventory, employees, customers, suppliers, billing, sales tracking, and customer feedback. The systemuses file handling for data storage, ensuring persistence, reliability, and reducedmanual work.

**System Features:**
 Login System – Secure access with username and password.
 Food Management – Add, display, search, delete, modify, and checkexpiryoffood items.
 Employee Management – Store and manage employee details (ID, name, role, salary).
 Customer Management – Manage customer details, including membershipowners.
 Supplier Management – Record supplier details and products supplied.
 Billing System – Generate bills with unique IDs, update stock, and save salesdata.
 Stock Overview – Display available stock with updated quantities.
 Sales Report – Generate detailed reports from billing history.
 Bill History – Retrieve past bills by ID.
 Feedback System – Customers can provide ratings and feedback for improvement.

**Advantages:**
 Automated system reduces manual errors
 User-friendly menu-driven interface
 Efficient stock and sales management  Useful reports for decision-making
 Simple and lightweight (C-based)

**Limitations:**
 No graphical interface (CLI only)
 Limited security (single hard coded login)
 No online database or cloud support  Prices are entered manually during billing.

**Topics covered:**
 C file handling
 Data storage arrays and structures
 Logic of comparison of dates
 Sorting algorithms
 Looping and conditional Chapters
 Modular programming with functions
