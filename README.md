# Grocery Shop Management System
A management system developed by C programming language in my first semester of undergraduation as a first project in this field

# Intro<br>
 Grocery Shop Management System, is a console- based C program designed to simplify the management of inventory, employees, customers, suppliers, billing, sales tracking, and customer feedback. The systemuses file handling for data storage, ensuring persistence, reliability, and reducedmanual work.

# System Features:<br>
 -Login System – Secure access with username and password.<br>
 -Food Management – Add, display, search, delete, modify, and checkexpiryoffood items.<br>
 -Employee Management – Store and manage employee details (ID, name, role, salary).<br>
 -Customer Management – Manage customer details, including membershipowners.<br>
 -Supplier Management – Record supplier details and products supplied.<br>
 -Billing System – Generate bills with unique IDs, update stock, and save salesdata.<br>
 -Stock Overview – Display available stock with updated quantities.<br>
 -Sales Report – Generate detailed reports from billing history.<br>
 -Bill History – Retrieve past bills by ID.<br>
 -Feedback System – Customers can provide ratings and feedback for improvement.

# Advantages:<br>
 -Automated system reduces manual errors<br>
 -User-friendly menu-driven interface<br>
 -Efficient stock and sales management  Useful reports for decision-making<br>
 -Simple and lightweight (C-based)

# Limitations:<br>
 -No graphical interface (CLI only)<br>
 -Limited security (single hard coded login)<br>
 -No online database or cloud support  Prices are entered manually during billing.
